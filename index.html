Projects

2024-: [FAPEMIG] Massive Graph Processing

This project aims to understand the performance challenges in large-scale graph processing, proposing new strategies and abstractions for massive processing systems. We will focus on pattern mining algorithms, machine learning, and artificial intelligence applied to graphs, with an emphasis on parallel and distributed environments. The research seeks to simplify the user experience with high-performance graph algorithms.
